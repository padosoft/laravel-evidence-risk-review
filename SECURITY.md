# Security Policy

## Supported Versions

Security fixes target the latest released major version.

## Reporting A Vulnerability

Do not open public issues for suspected vulnerabilities.

Send a private report to the repository maintainers with:

- affected version or commit
- reproduction steps
- impact assessment
- any suggested mitigation

The package is designed with default-OFF LLM, HTTP, MCP, and persistence surfaces. Reports that identify bypasses of those defaults are treated as high priority.
